# App gif's en Angular

Proyecto de gif's en Angular

## Link del proyecto

<https://app-gifs-angular-skierhy.netlify.app/>

## Instrucciones para el proyecto

Hacer el build del proyecto
`npm run build`

Correr el proyecto
`npm run start`

## Tecnologías utilizadas en este proyecto

- Angular
- bootstrap
